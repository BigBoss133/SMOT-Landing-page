# Audit 2026-05-12

## Stato attuale
- Sito statico in singolo `index.html` (≈1.7 MB) con inline CSS/JS e grafica complessa
- Nessuna toolchain (build, lint, minify) configurata
- Accessibilità limitata: assenza di skip-link, headings gerarchici irregolari, immagini senza `alt`
- Nessuna automatizzazione deploy o pipeline QA

## Miglioramenti consigliati
1. Estrarre assets in componenti modulari (ad es. Astro/Vite + Tailwind)
2. Minificare immagini e adottare `loading="lazy"` per media pesanti
3. Aggiungere meta tag SEO, OpenGraph, e file `sitemap.xml` generato automaticamente
4. Definire workflow GitHub Actions: build (es. `npm run build`), HTML lint (pa11y), deploy Netlify
5. Creare versione responsive testata a 320 px, 768 px, 1440 px

## Prossimi passi
- Introdurre toolchain (p.es. Vite + PostCSS) salvando HTML sorgente in `src/`
- Aggiungere suite di test accessibilità (axe-core) e screenshot testing
- Documentare nel README istruzioni di build/deploy
